## Get Started

- Clone git repo.
- Create python environment

```bash
python venv -venv
```

- Initialise environment

```bash
venv\scripts\activate
```

- Install python libraries in requirements.txt

```bash
pip install -r requirements.txt
```

- Run python app

```bash
app/app.py
```
